Mon projet est un générateur de cartes d'identité musicales interactives. Ce projet permet de recuperer, collectionner et sauvegarder des "badges" de profil personnalisés, inspirés par l'interface de Spotify et l'univers des cartes à collectionner type Pokemon.



**À propos**

L'objectif est simple : proposer une expérience ludique où l'utilisateur peut transformer ses goûts musicaux en un visuel ludique et agréable.

L'application permet de chercher des artistes parmi une base de données API (sur mon spotify ou j'ai crée la plalist de 100 sons), d'y associer un Artiste, et de générer une carte stylisée qui reprend les codes visuels du streaming et des cartes Pokemon.


**Fonctionnalités**

Recherche Artistes : Explorer via l'API Spotify pour trouver sa carte a collectionner.

Système de Collection : Sauvegarde des badges créés dans une galerie (Json).

Filtres : Possibilité de filtrer sa collection des 100 sons pour retrouver facilement un artiste ou badge.

Design : Interface inspirée des cartes à collectionner Pokemon et du mode sombre de Spotify.


**Outils utilisés**

API : Spotify Web API

Outils : [Golang, HTML, CSS, JSON]

Spotify : Création d'une playlist sur mon compte + récuperation des données
